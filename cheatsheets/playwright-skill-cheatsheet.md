# Playwright Skill Cheatsheet

## Common Prompts

- `用 playwright 打开这个网站并截图`
- `用 playwright 登录这个页面，然后抓取页面主要内容`
- `用 playwright 检查这个网页在移动端有没有布局问题`
- `用 playwright 点击这个按钮并观察页面变化`
- `用 playwright 抓取这个页面的标题、价格和链接`

## Login / Form Fill

- `用 playwright 打开登录页，输入账号和密码，点击登录，然后截图结果`
- `用 playwright 填写这个表单，并检查提交后是否成功`
- `用 playwright 逐步执行这个网页操作流程，并告诉我每一步结果`

## Data Extraction

- `用 playwright 提取这个页面的全部商品标题和价格`
- `用 playwright 抓取这个列表页面的所有链接`
- `用 playwright 提取这个后台页面里的表格数据`

## UI / QA

- `用 playwright 检查这个网页的按钮、输入框和导航是否正常工作`
- `用 playwright 看这个页面有没有明显报错或加载失败`
- `用 playwright 以移动端视角检查这个页面的布局问题`

## Combined Workflow

- `用 playwright 打开这个网站，等待页面加载完成，抓取主要内容，再截图保存`
- `用 playwright 登录后台，进入指定页面，提取关键信息，并给我截图`

## Suggested Pattern

1. 打开页面
2. 等待加载
3. 抓取内容或执行点击
4. 截图留档
5. 总结结果
