# Workflow Overview

## Daily use

### News and summaries

- Use `cheatsheets/news-prompts-cheatsheet.md` with `newstoady`
- Convert summaries into Feishu or Telegram friendly formats

### Browser automation

- Use `cheatsheets/agent-browser-cheatsheet.md` for direct browser CLI work
- Use `cheatsheets/playwright-skill-cheatsheet.md` when working through the Playwright skill

### CLI operations

- Use `cheatsheets/github-cli-cheatsheet.md` for GitHub tasks
- Use `cheatsheets/feishu-cli-cheatsheet.md` for Feishu/Lark tasks

## Skill management

- Use `scripts/find-skills.py` to inspect installed skills
- Use `scripts/skill-vetter/check_skill.py` to validate a skill before sharing or reusing it
- Review `docs/skill-vetter-report.md` for the latest validation snapshot

## Suggested repo usage

Add new items according to type:

- Put short references in `cheatsheets/`
- Put reusable scripts in `scripts/`
- Put generated reports and notes in `docs/`
- Put skill definitions or exports in `skills/`
